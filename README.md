# cables-td

##  English

Compilation of cable properties for transmission &amp; distribution calculations.

Suppose you landed here with the usual question: what´s the resistence for cable X? And the linear mass for cable Y. And so on. So I gathered some info from a number of references here.

Of course this is a living reference, feel free to complete and correct.

The repo is divided by category and a separate file for material properties.

Please all units SI.

## Português

Compilação de propriedades de cabos para estudos de transmissão e distribuição.

A repositório será composto por uma série de arquivos markdown, para cada família de cabos condutores.
